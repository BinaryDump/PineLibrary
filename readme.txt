PineLibrary est un outil qui permet la gestion de prêt de livres. Pour cela l'utilisateur doit s'inscrire et ajouter ses livres dans la base de données (PostgresSql).


Projet réalisé lors de notre BAC 2 en Informatique de Gestion à la HELHa de Montignies-Sur-Sambre (Belgique)
Ce projet a été réalisé dans le cadre du cours de JAVA.
